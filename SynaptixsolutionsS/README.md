# Synaptix Solutions Ltd - Full Stack Website

## Project Structure

```
synaptix-website/
├── README.md
├── frontend/          # Source files (for editing)
│   ├── index.html
│   ├── booking.html
│   ├── admin.html
│   ├── style.css
│   └── script.js
└── backend/           # Render deployment
    ├── package.json
    ├── server.js
    ├── .gitignore
    ├── README.md
    └── public/        # Static files served by Express
        ├── index.html
        ├── booking.html
        ├── admin.html
        ├── style.css
        └── script.js
```

## Setup

1. Copy `frontend/*` files into `backend/public/`
2. Deploy `backend/` folder to Render
3. Add environment variables in Render dashboard
